Personal Settings - Help

 

User Self Administration or Personal Profile Page allows each user to Change their Password, Setup and Auto Reply Message, and to setup an e-mail address where their mail should be forwarded to.

After clicking on the Personal Administration Page, user will be presented with a Login Dialogue box;

Please enter your e-Mail user ID and password and click on to proceed. Please note that the Self Administration pages are Java based and load slower then normal web pages. Please be patient while waiting for server response.
After login, the Personal Profile page is displayed. Following picture points out the important parts and controls on this page.
Password can be changed on the first page shown after login. Full Name is the name that is shown in the e-mail as the name of the sender. This does not effect users e-mail address. Be aware that Abuse of this facility will result in community service sentences including but not limited to cleaning up the smoking area and washing dishes.
Click on to setup Auto Reply or e-mail forwarding. Please note that the server does not keep copies of forwarded e-mails. Therefore, if you set a forwarding e-mail address, all e-mail will be forwarded to that e-mail address and then deleted off the server.
